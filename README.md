# CMS Admin Panel

This is the Admin Panel for managing my Custom CMS based Portfolio.

Using this panel, the admin can add, update and delete all portfolio content.

---

## Features

- Admin Login
- Manage About Section
- Manage Skills
- Manage Projects
- Manage Blogs
- Manage Experience
- Manage Testimonials
- Manage Services
- View Contact Messages
- Upload Images

---

## Technologies Used

- HTML
- JavaScript
- React (CDN)
- Tailwind CSS
- Fetch API

---

## How to Run

1. Open `index.html` using Live Server in VS Code  
2. Make sure backend is running on:

http://localhost:5000
